# Routing rules

## Route away (do not troubleshoot here)
Set **Category = ROUTE** when the notes indicate any of:

- **Epic** (clinical system) involvement
- **Apogee printing** / managed print queues owned by Apogee team
- **Network** faults (site-wide/Wi-Fi/VLAN/switching/routing, confirmed network outage)
- **Telephony** issues (handsets, call manager, telephony queues)
